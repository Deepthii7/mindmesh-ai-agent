# 📚 StudyMate AI

StudyMate AI is a multi-agent educational assistant designed to help students learn more effectively through concept explanations, quiz generation, and personalized study planning.

## 🎯 Problem

Students often spend significant time searching for explanations, creating study plans, and preparing practice questions. Learning resources are scattered across multiple platforms, making studying less efficient.

## 💡 Solution

StudyMate AI brings learning assistance into one place by providing:

- Clear concept explanations
- Automatically generated quizzes
- Personalized study plans
- AI-powered learning support

## 🤖 Multi-Agent Architecture

StudyMate AI consists of multiple specialized agents:

### Explainer Agent
Provides beginner-friendly explanations for topics.

### Quiz Agent
Generates practice questions and quizzes.

### Study Planner Agent
Creates personalized learning roadmaps and study schedules.

### Coordinator Agent
Routes user requests to the appropriate agent.
